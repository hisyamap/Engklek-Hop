This game was developed to built a learning environment where users can improve their english grammar proficiency by playing the game.

The Dialogic addon was specifically chosen to handle the in-game educational content, including interactive quizzes and step-by-step player tutorials.

You can play in your browser here [https://digidogames.com/](https://engklekhop.digidogames.com/)
